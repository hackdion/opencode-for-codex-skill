# v0.1.0

## What This Release Includes

This is the first public release of `opencode-for-codex`.

Included in this release:

- the reusable `SKILL.md` for Codex
- a portable shell wrapper for archived OpenCode consultations
- a lightweight self-check script
- operational notes for provider and TLS-related failure handling
- GitHub community health files and collaboration templates

## Intended Use

Use this project when Codex needs a traceable second opinion from OpenCode and you want to keep the prompt, raw output, and final Codex decision archived near the target workspace.

## Known Limitations

- real OpenCode availability still depends on external provider health
- maintainers should replace repository URL placeholders before or immediately after publishing
- the repository currently focuses on a shell-first workflow rather than multi-platform packaging
