# Support

## Where To Ask What

- Bug reports: open a GitHub issue using the bug template
- Feature ideas: open a GitHub issue using the feature template
- Usage questions: open a GitHub discussion or issue titled `question: ...` until a dedicated discussion space exists
- Security concerns: open a GitHub issue and follow [SECURITY.md](SECURITY.md)

## What To Include

When asking for help, please include:

- what you were trying to do
- the exact command you ran
- relevant output from `stderr.log`, `status.txt`, or `answer.md`
- your environment details if they matter
- what you already tried

## Maintainer Scope

This repository is maintained as a practical open source utility, not a guaranteed support service. Clear reports with reproducible evidence will get the best response.

## Contact Path

This project currently uses GitHub issues as the only support and security contact path.

- GitHub issues for normal support
- GitHub issues for feature requests
- GitHub issues for security reports, without posting secrets
